# Landing Page

هذه صفحة هبوط مستقلة تماما عن مشروع الـ POS.

## الملفات

- `index.html`: الصفحة الرئيسية
- `styles.css`: التصميم الكامل
- `script.js`: إعداد رقم واتساب ورابط التحميل

## ما الذي يجب تعديله قبل النشر؟

افتح `script.js` وعدل هذه القيم:

- `whatsappNumber`
- `whatsappDisplay`
- `whatsappMessage`
- `downloadUrl`
- `downloadLabel`

## وضع ملف البرنامج

إذا كنت تريد أن يكون التحميل من نفس الموقع:

1. أنشئ مجلدا باسم `downloads`
2. ضع داخله ملف التثبيت مثل:
   `System-De-Vente-Setup.exe`
3. اترك `downloadUrl` كما هو أو عدله حسب اسم الملف

## النشر

يمكنك نشر الصفحة على أي استضافة static مثل:

- Netlify
- Vercel
- GitHub Pages
- استضافة cPanel عادية

إذا كان ملف التثبيت كبيرا، من الأفضل رفعه في خدمة منفصلة ثم وضع رابطه في `downloadUrl`.
"# pos-system" 
"# System-De-Vente"
